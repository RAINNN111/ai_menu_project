# AIケーキ注文システム

## 概要

Djangoを使用して開発したAI対応ケーキ注文システムです。
ユーザー登録・ログイン、ケーキ注文、カスタマイズ機能、クーポン機能などを実装しました。

## 主な機能

* ユーザー登録 / ログイン
* ケーキメニュー表示
* カート機能
* 注文機能
* クーポン機能
* お気に入りフレーバー登録
* AIケーキカスタマイズ
* ユーザー好みに基づくおすすめ表示

## 使用技術

* Python
* Django
* SQLite
* HTML / CSS / JavaScript
* Fetch API (Ajax)
* Render (デプロイ)
* Git / GitHub
